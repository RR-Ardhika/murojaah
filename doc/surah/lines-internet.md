## Methodology Notes

1. **Line Types**: The mushaf-layout data includes ALL line types:
   - `surah-header`: Surah name/title line (e.g., "سورة البقرة")
   - `basmala`: Bismillah line (except for Surah At-Taubah)
   - `text`: Quran text lines

2. **Counting Method**: Each line on a page is counted as 1 line, regardless of type.

3. **Page Structure**: Standard 15 lines per page, but some pages may have fewer due to:
   - Surah headers
   - Bismillah lines
   - End of Quran (page 604)

## Other Sources Checked (No Line Counts Available)

The following sources were checked but **do not provide per-surah line counts**:

| Source | What It Has | Why Not Useful |
|--------|-------------|----------------|
| [quran.com API](https://api.quran.com/api/v4/chapters) | Page ranges, ayah counts | No line counts |
| [alquran.cloud](https://api.alquran.cloud/v1/surahs) | Surah metadata | No line counts |
| [quran-meta](https://github.com/quran-center/quran-meta) | Juz, hizb, page metadata | No line counts |
| [corpus.quran.com](https://corpus.quran.com/) | Word grammar, syntax, morphology | Linguistic focus, no layout data |
| Islam Stack Exchange | Total lines only (~8,815) | No per-surah breakdown |

**mushaf-layout is the only comprehensive source** that provides:
- Per-page JSON files with line-by-line breakdown
- Line type classification (header, basmala, text)
- Complete 604-page coverage
